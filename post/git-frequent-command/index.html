Git 常用命令

2019-12-05 1 min read # Git

拉取远端指定分支

git clone -b [branch name] [git url] [local folder name]

分支

下载远程仓库的所有变动

git fetch

列出所有远程分支

git branch -r

拉取 xx 分支并合并到本地分支

git pull origin [branch name]

切换到指定分支，并更新工作区

git checkout [branch name]

切换到上一个分支

git checkout -

查看信息

显示有变更的文件

git status

显示暂存区和工作区的差异

git diff [filename]

撤销

恢复暂存区的指定文件到工作区

git checkout [filename]

恢复暂存区的所有文件到工作区

git checkout .
