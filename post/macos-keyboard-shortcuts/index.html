macOS 快捷键

macOS 功能描述
⌃ + a/e 将光标移动到行首或行末
⌃ + Power 显示一个对话框，询问重新启动、睡眠还是关机
⇧ + ⌃ + Power 休眠显示器
⌘ + a 全选
⌘ + o 打开对话框选择要打开的文件
⌘ + p 打印当前文稿
⌘ + q 退出当前程序
⌘ + v 将剪贴板内容粘贴到当前文本中
⌘ + x 移除选中的内容，然后将副本放在剪贴板中
⌘ + ， 打开当前应用的偏好设置
⌘ + ⇥ 在打开的应用中切换到下一个最近使用的应用
⌘ + Space 打开搜索栏
⌥ + ⌘ + l 打开下载列表
⌥ + ⌘ + esc 强制退出应用
⌥ + ⌘ + ←/→ 切换标签页
⌥ + ⌘ + Power 休眠 Mac
⌘ + f 搜索当前文件
⌃ + ⌘ + f 当前窗口最大化
⌘ + h 隐藏应用当前窗口
⌥ + ⌘ + h 查看最前面的应用但隐藏所有其他应用
⌘ + m 最小化当前窗口
⌥ + ⌘ + m 最小化当前应用所有窗口
⌘ + n 创建新窗口
⇧ + ⌘ + n 创建一个智能窗口
⌘ + r 刷新
⇧ + ⌘ + r 强制刷新 / 打开 AirDrop 文件夹
⌘ + s 存储当前操作
⇧ + ⌘ + s 另存当前操作
⌘ + t 打开新标签页
⇧ + ⌘ + t 打开最近关闭的标签页
⌘ + w 关闭当前标签页
⌥ + ⌘ + w 关闭应用当前窗口
⌘ + z 撤销上一个命令
⇧ + ⌘ + z 取消撤销
